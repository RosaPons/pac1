BABEL

Que és

És una eina que ens permet transformar (transpilar) el nostre codi JS d’última generació a JS que qualsevol navegador o versió de Node.js entengui.

Per part de les empreses dels navegadors, tarden molt mesos i fins i tot anys en implementar les noves versions de ES (Ecma Script). Amb Babel podrem utilitzar el més nou de JS amb la tranquil·litat que funcionarà en la majoria de navegadors sense problemes.

Babel funciona amb plugins o presets (conjunt de plugins) per indicar quines són les coses que volem transformar.

Com utilitzar Babel

Crearem el nostre projecte amb la següent ordre:

$ npm init

Instal·lem Babel amb la següent ordre:

$npm install –save-dev babel-cl

Executarem la següent ordre per afegir un dels presets per Babel:

$npm install babel-preset-env --save-dev

En el directori arrel del nostre projecte crearem un nou fitxer .babelrs i és on hi desarem la configuració de Babel.

Editarem el fitxer .babelrs i afegirem la següent linia:

{ “presets”: [“es2015”] }

Amb això Babel transpilarà el nostre codi JS a JS ES5.

Finalment executarem $npm run build per tal de que Babel faci la transformació.
