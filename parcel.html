NPM

Que és

Npm és l'administrador de paquets que ve preinstalat amb la plataforma de servidor Node.js.

S'utilitza per instal·lar programes de Node des d’el registre Npm. És una eina que ajuda als desenvolupadors a instal·lar, eliminar i administrar programes de Node de tercers, bàsicament les dependències utilitzades en un programa.

Cóm instal.lar NPM

Instal·lar Npm és simple, atès que s'instal·la automàticament quan instal·la Node.js. No et preocupis, Node.js tampoc és tan difícil d'instal·lar.

En un navegador web, visita nodejs.org, i fes clic en el botó d'instal·lació.

Per comprovar la versió instal·lada executa:

$ node –version

Com instal·lar paquets/móduls

Executa des de la línia de comandos en el directori on es desitja instal·lar el paquet:

$ npm install nom_paquet

(*) El paràmetre -g davant nom del mòdul, instal·la el mòdul de forma global, la qual cosa fa disponible en qualsevol des de qualsevol ubicació o projecte.

Per desinstal·lar:

$ npm uninstall nom_paquet

Com iniciar una projecte amb Npm

Executa des de la línia de comandos en el directori on es desitja crear el projecte:

$ npm init -y

(*) El paràmetre -g davant nom del mòdul, instal·la el mòdul de forma global, la qual cosa fa disponible en qualsevol des de qualsevol ubicació o projecte.

Per desinstal·lar:

$ npm uninstall nom_paquet
