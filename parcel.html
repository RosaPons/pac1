PARCEL

Que és

És un empaquetador de aplicacions web molt valorat per la comunitat de desenvolupadors.

És molt ràpid i no necessita configuració.

Parcel te un servidor de desenvolupament encastat que s'encarrega de refer la nostra aplicació en calent a mesura que fem canvis.

Cóm instal.lar Parcel

Utilitza Npm o Yarn per instal.lar Parcel en el teu projecte.

$npm install parcel-bundler --save-dev

o

$yarn add parcel-bundler --dev

Com iniciar un projecte

Executa la següent ordre:

$npm init -y

o

$yarn init -y

Ordre per activar el servidor: $parcel index.html

Des d'el navegador executa http://localhost:1234/ per visualitzar les pàgines que estàs desenvolupant.

En modus desenvolupament / producció

En el fitxer package-json , afegeix aquests scripts:

{

"scripts": {

"dev": "parcel " {

"build": "parcel build " {

      }

}

Para executar-lo en modus desenvolupament:

$yarn dev

o

$npm run dev

Para executar-lo en modus producció:

$yarn dev

o

$npm run dev
