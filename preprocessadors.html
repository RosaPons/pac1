Preprocessadors

Els preprocessadors són un conjunt d'eines que reben arxius escrits en diversos llenguatges i els converteixen en codi comprensible pel navegador. A vegades també són anomenats compiladors o transpiladors.

Els navegadors només són capaços de processar i executar HTML, CSS i JavaScript. Qualsevol altre llenguatge ha d’acabar convertit en un d’aquests.

Els preprocessasdors que generen CSS s’utilizen per afegir noves funcionalitats com l’ús de variables o imbricació. Aquí destaquem PostCSS.

Els preprocessadors que generen JavaScript són usats per transformar llenguatges completament diferents a JavaScript, i també per transformar versions modernes del llenguatge en versions aptes per a tots els navegadors. Babel ens ho permet.

Existeixen altres preprocessadors tant per CSS i JS com per HTML; TypeScript per JavaScript; SASS/SCSS per CSS o PUG per HTML.
