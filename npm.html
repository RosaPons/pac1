NPM

Que és

Npm és l'administrador de paquets que ve preinstalat amb la plataforma de servidor Node.js.

S'utilitza per instal·lar programes de Node des d’el registre Npm.

És una eina que ajuda als desenvolupadors a instal·lar, eliminar i administrar mòduls Node de tercers i les seves dependències.

Cóm instal.lar NPM

Instal·lar Npm és simple, atès que s'instal·la automàticament quan instal·la Node.js.

En un navegador web, visita nodejs.org, i fes clic en el botó d'instal·lació.

Per a comprovar la versió instal·lada executa la següent ordre:

$ node –version

Com iniciar un projecte Npm

Executa la següent ordre des d'el directori on es desitja crear el projecte:

$ npm init -y

Ens crearà de forma automàtica el fitxer package.json que conté informació del nostre projecte i les seves dependencies.

Les dependències que anem instal.lant s'afegiran en la captera node_modules.

Com instal·lar hi paquets

Executa la seguënt ordre des de la línia de comandos i des d'el directori on es desitja instal·lar el paquet:

$ npm install nom_paquet

(*) El paràmetre -g davant nom del mòdul, instal·la el mòdul de forma global, i el fa disponible per qualsevol ubicació o projecte.

Per desinstal·lar:

$ npm uninstall nom_paquet
