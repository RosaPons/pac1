Com utilitza i instal.ar el producte

POSTCSS

Que és

PostCSS no és un generador de codi CSS.

El que fa és proporcionar un analitzador de codi CSS i un conjunt de complements que permeten analitzar, optimitzar i transformar el codi CSS.

Actualment, hi ha més de 200 complements per a PostCSS.

Cada complement s’instal·la per separat i permet crear-ne de propis.

Per a més informació clica aquí.

Complement Autoprefixer

Autoprefixer analitza arxius CSS i agrega prefixes de proveïdor a les regles CSS utilitzant la base de dades CanIUse.

Exemples de la transformació.

Codi original:

a {

"transition: transform 1s

}

Codi transformat per Autoprefixer:

a {

-webkit-transition: -webkit-transform 1s;

transition: -ms-transform 1s;

transition: transform 1s {

}

Per a més informació clica aquí.

Cóm instal.lar i executar Autoprefixer

Des de Npm executa la següent ordre:

$ npm install –save-dev autoprefixer

Crea l'arxiu .postcssrc a l’arrel del projecte i afegeix les següents línies

a {

“plugins”: { “autoprefixer”: true }

}

Executa la següent ordre i Autoprefixer aplicarà la transformació.

$ npm run build
